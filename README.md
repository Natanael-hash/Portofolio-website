### Running Locally

1. Clone the repository:
   ```bash
   git clone [https://github.com/Natanael-hash/Portofolio-website.git](https://github.com/Natanael-hash/Portofolio-website.git)
   ```
2. Navigate to the project directory:
   ```bash
   cd Portofolio-website
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   # yarn install
   ```
4. Start the development server:
   ```bash
   npm run dev
   # or
   # yarn dev
   ```
5. Open [http://localhost:5173](http://localhost:5173) (sau portul afiÈ™at de Vite) in your browser.
